# QZStock - 股票策略回测工具

![Python](https://img.shields.io/badge/Python-3.7+-blue.svg)
![customtkinter](https://img.shields.io/badge/GUI-customtkinter-purple.svg)
![License](https://img.shields.io/badge/License-MIT-green.svg)

QZStock 是一个基于 Python 和 customtkinter 的股票策略回测工具，采用深色主题界面，支持多技术指标策略回测和可视化分析。

## ⚠️ 重要声明

**该软件仅供学习和娱乐使用，不构成任何投资建议。策略胜率较低，请勿用于实际交易！**

## ✨ 功能特点

1. **多指标策略**：整合 EMA、MACD、ADX、RSI、量比等多种技术指标
2. **参数可调**：通过滑块实时调整策略参数，即时查看回测效果
3. **可视化展示**：净值曲线、股票价格、月度收益柱状图多维度展示
4. **交易信号**：清晰展示最近交易信号和策略统计数据
5. **数据缓存**：自动下载并缓存股票历史数据（CSV格式）
6. **多线程处理**：后台任务队列，不阻塞UI界面
7. **深色主题**：GitHub Dark 风格的现代化界面设计

## 🛠️ 技术栈

| 类别 | 技术 |
|------|------|
| GUI框架 | customtkinter (基于Tkinter的现代化UI库) |
| 数据分析 | Pandas, NumPy |
| 可视化 | Matplotlib |
| 数据源 | AKShare (新浪财经接口) |
| 多线程 | threading, queue |
| 主题风格 | NocturneTheme (深色主题) |

## 📦 安装与使用

### 1. 安装依赖库

```bash
pip install customtkinter akshare pandas numpy matplotlib
```

### 2. 运行程序

```bash
python QZstock.py
```

### 3. 使用步骤

1. 点击「输入代码」按钮，输入6位股票代码
2. 程序自动下载最近5年的日线数据（前复权）
3. 数据加载完成后自动执行策略回测
4. 通过左侧滑块调整参数，点击「执行策略」重新回测
5. 点击「重置参数」可恢复默认参数

## ⚙️ 参数说明

### EMA 周期

| 参数 | 默认值 | 范围 | 说明 |
|------|--------|------|------|
| 短周期 | 5 | 3-20 | 短期EMA均线周期 |
| 中周期 | 13 | 10-40 | 中期EMA均线周期 |
| 长周期 | 30 | 20-60 | 长期EMA均线周期 |

### MACD 参数

| 参数 | 默认值 | 范围 | 说明 |
|------|--------|------|------|
| 快线 | 9 | 5-20 | MACD快线周期 |
| 慢线 | 21 | 12-40 | MACD慢线周期 |
| 信号周期 | 6 | 3-14 | MACD信号线周期 |

### 风险控制

| 参数 | 默认值 | 范围 | 说明 |
|------|--------|------|------|
| 止损比例 | 30% | 0-50% | 最大止损比例 |
| 量比阈值 | 0.5 | 0.1-3.0 | 成交量与20日均量的比值阈值 |

### 趋势过滤

| 参数 | 默认值 | 范围 | 说明 |
|------|--------|------|------|
| ADX 阈值 | 20 | 10-40 | ADX趋势强度过滤阈值 |
